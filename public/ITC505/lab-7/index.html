Purpose: This project is a space-themed Mad Lib game built using ExpressJS and front-end web technologies.

Technology Stack: Node.js, ExpressJS, vanilla JS, HTML, and CSS.

User Interface Theme: Futuristic space design with dark backgrounds and glowing text.

Font Choices: Sans-serif fonts like Segoe UI / Verdana for readability and clean aesthetics.

Interactive Features: Random word buttons for each field using in-memory JavaScript dictionaries.

Validation: Server-side validation ensures all fields are filled before submission.

Deployment: Runs locally on port 8080 and is also deployed live using Render.

Learning Outcome: Gained experience with ExpressJS, form handling, deployment, and enhancing user interactivity.
