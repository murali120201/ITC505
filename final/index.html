Project Overview:
In this project, I developed a playable version of the classic Lights Out puzzle game using HTML, CSS, and JavaScript. The primary goal was to create a dynamic and interactive game board that automatically initializes with a random but solvable puzzle configuration. When the user successfully solves the puzzle, the game provides clear feedback through a celebratory popup message.

Core Game Logic:
Clicking a cell on the 5x5 board toggles its own state along with its adjacent neighbors (up, down, left, right). Instead of assigning random states, the board is initialized using a simulated series of valid random moves, which guarantees that the configuration is solvable.
